frontend-nanodegree-arcade-game

# FROGGER
This project has been completed as a part of the FrontEnd Nanodegree Program requirements.

## Goal of the Game
The player has to reach the water blocks without losing it's lives.

## Instructions to play
* The player can move left,right,up and down with the respective arrow keys.
* The player should not collide with the bug. 
* If collision occurs, a life and 50 points are lost and the player returns to the starting position.
* Reach the water block without hitting any bugs to win the best score.

## Developing the Game
HTML,CSS,Javascript emphasizing on object-oriented features was used to develop this arcade game.
Google Fonts was added to this game.
